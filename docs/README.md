## 📁 sistemas

**Carpeta:** `sistemas/`

**Contenido:** Documentación técnica del proyecto TechManage 1.0.

**Subcarpetas:**
- `evidencias/` - Evidencias del desarrollo del proyecto
  - `evidencias.md` - Documento principal de evidencias

**Documentos:**
- `analisis_datos.md` - Análisis de datos del sistema
- `diagrama_ER.drawio` - Diagrama entidad-relación
- `modelo_relacional.drawio` - Modelo relacional de la base de datos

**Descripción del proyecto:**

TechManage 1.0 es una aplicación de gestión para tiendas de componentes informáticos. Permite administrar productos, clientes, proveedores, tiendas, ventas e inventario mediante una interfaz gráfica JavaFX.

**Tecnologías utilizadas:**
- Java + JavaFX (interfaz gráfica)
- MySQL (base de datos)
- JAXB (serialización XML)
- Maven (gestión de dependencias)
- Git + GitHub (control de versiones)

**Estado actual:** ✅ Completado (Fase 5)
